# Music Store Sales & Customer Analysis

## Business Problem
Chinook is a digital music store selling music tracks across multiple countries.
The goal of this project is to analyze sales data to understand revenue trends,
customer behavior, and music preferences.

## Dataset
Chinook Sample Database (SQLite)

## Tools Used
- SQL (SQLite)
- DB Browser for SQLite
- Power BI

## Analysis Performed
- Total revenue calculation
- Country-wise revenue analysis
- Top customers by spending
- Genre popularity analysis
- Artist-wise revenue analysis

## Dashboard
An interactive Power BI dashboard was created to visualize KPIs, revenue trends,
top customers, popular genres, and top artists.

## Outcome
The analysis revealed that revenue is concentrated in a few countries, a small
number of customers contribute significantly to sales, and certain genres and
artists dominate customer preferences.
